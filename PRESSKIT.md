# Crackveil Vanguard Press Kit

## 30 字 Pitch

五人裂隙小隊割草生存：契約改規則、詞綴精英破節奏、武器進化與殘響 Meta 推你再開一局。

## 線上網址

https://mars-tw.github.io/crackveil-vanguard/

## 操作

- `WASD` / 方向鍵：移動。
- `P` / `Esc`：暫停、設定、複製種子。
- 行動裝置：左下虛擬搖桿。
- 開局契約畫面：可輸入或貼上種子開新局。

## 特色

- 三支柱戰鬥：小隊站位、自動武器、敵潮壓力。
- 裂隙契約：開局選一條本局規則，改變風險與收益。
- 詞綴精英：裂殖、磁滯、疾閃三種規則怪，具備色彩與形狀提示。
- 武器進化：投入武器後解鎖質變卡，形成第二層 build。
- 殘響 Meta：結算帶回碎片，解鎖契約槽與小幅永久整備。
- 本地成就與種子分享：結算追蹤本局解鎖，玩家可複製同一局種子互相比拼。

## 瀏覽器需求

- Godot 4.7 單執行緒 WebAssembly build。
- 建議使用最新版 Chrome、Edge、Firefox 或 Safari。
- 不需要 SharedArrayBuffer；適合 GitHub Pages 等靜態站台。
- 首次播放音效需要一次點擊或按鍵以解鎖瀏覽器音訊。
